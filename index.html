Aquobot

I've written written a bot for the online chat room service Discord. Aquobot can perform various functions based on user query, as well as respond to certain keywords. Aquobot is currently active and is being used in several servers. The source code is located here or you can click the link in the menu to view its command list.


Kirby Randomizers

I've written some programs that will randomize various things in three different Kirby games: Kirby's Adventure, Kirby's Nightmare in Dream Land, and Kirby and the Amazing Mirror. Some features include changing Kirby's color, changing the level order, changing the abilities enemies give you, and more! If you want to check it out, visit my GitHub, or check out the link in the menu above.


Ecco the Dolphin Text Generator

One of my true obsessions in this world is the Sega Genesis game Ecco the Dolphin. To that end, I've made a tool that lets you generate text in the style from the game. Just click the link in the menu above to see it for yourself!
