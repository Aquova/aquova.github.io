Aquobot

I've written written a bot for the online chat room service Discord. Aquobot can perform various functions based on user query, as well as respond to certain keywords. Click the image to view the source code, or you can click the link in the menu to view the command list.


Kirby Randomizers

I've written some programs that will randomize various things in three different Kirby games: Kirby's Adventure, Kirby's Nightmare in Dream Land, and Kirby and the Amazing Mirror. Some features include changing Kirby's color, changing the level order, changing enemy abilities, and more!


Ecco the Dolphin Text Generator

One of my true obsessions in this world is the Sega Genesis game Ecco the Dolphin. To that end, I've made a tool that lets you generate text in the style from the game.

@WUSTL_Weather
