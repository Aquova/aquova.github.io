Aquobot

I'm currently writing a bot for the online chat room service Discord. Aquobot can perform various functions based on user query, as well as respond to certain keywords. Aquobot is currently active and is being used in 4 servers. The source code is located on GitHub or you can click the link in the menu to view its command list.

Kirby's Adventure Randomizer

I've written a Python program that alters a copy of Kirby's Adventure for NES, randomizing things about the game, such as which abilities you gain and what color your character is. If you want to check it out, visit my GitHub.
